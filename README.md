leaderf-digraph
================

A leaderf source of digraph.

![screenshot]

Usage
-----

``` {.vim}
inoremap <C-k> <C-o>:Leaderf digraph<CR>
```

Or like \[chrisbra/unicode.vim\]

``` {.vim}
inoremap <C-x><C-z> <C-o>:Leaderf digraph<CR>
```
